Qu’est-ce que l’Open Source ?

L’open source désigne des logiciels dont le code source est accessible et modifiable librement. Tout le monde peut : lire, améliorer, modifier et partager le code selon la licence.

Pourquoi utiliser l’Open Source ?

  • Liberté : tu choisis ce que tu veux faire avec le logiciel.
  • Transparence : le code est public, donc facile à vérifier et améliorer.
  • Communauté : des milliers de personnes collaborent pour corriger et améliorer le logiciel.
  • Prix : souvent gratuit, mais surtout le contrôle appartient à l’utilisateur.

Exemples connus d’outils Open Source :

  • Systèmes : Linux (Ubuntu, Fedora)
  • Navigateurs : Firefox
  • Retouche photo : GIMP, Krita
  • Montage vidéo : Kdenlive, Blender
  • Développement : Git, VS Code (partiellement open source)
